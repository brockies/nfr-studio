# GDPR (Ecommerce) Notes (Anonymized)

This is a lightweight reminder document intended as a knowledge base source, not legal advice.

## Common Ecommerce Requirements Patterns
- Data minimisation and retention: define retention windows for accounts, orders, logs, and analytics.
- Lawful basis for processing and consent flows for marketing.
- Data subject rights (DSAR): export and deletion workflows with auditability.
- Breach notification process and incident response runbooks.
- Processor/sub-processor inventory and DPIA where appropriate.
